 📊 Sales Performance Dashboard

📌 Project Overview
This project is a **Sales Performance Dashboard** designed to track and analyze key business metrics such as sales, profit, and total orders.

The dashboard provides insights into **regional performance, category-wise sales, and geographical distribution**, helping businesses make data-driven decisions.


🎯 Objectives
- Analyze overall sales performance
- Track profit and order trends
- Identify top-performing regions and categories
- Visualize geographical sales distribution

 📊 Key Metrics (KPIs)
- 💰 Total Sales: $2,261,537  
- 📈 Total Profit: $452,307  
- 📦 Total Orders: 9,800  

🛠️ Tools & Technologies Used
- Tableau 
- Data Visualization
- Data Analysis

📈 Features
- KPI Cards for quick insights
- Interactive charts for category analysis
- Region-wise performance comparison
- Geographical sales distribution map

📂 Project Files
- `Sales Performance Dashboard.twbx` 
- train.csv
- `README.md`

🚀 How to Use
1. Download the dashboard file
2. Open using Tableau 
3. Explore insights using filters and visuals

💡 Insights Gained
- Certain regions contribute more to overall sales
- Profit varies significantly across categories
- Sales distribution is not uniform geographically
